# 🏦 Loan Default Risk Assessment System

A machine learning project that predicts whether a customer is likely to default on a loan using **Logistic Regression implemented from scratch with NumPy**. The project also includes **Linear Regression**, data preprocessing, model evaluation, and an interactive prediction system.

## 🚀 Features

- Logistic Regression from Scratch (NumPy)
- Linear Regression from Scratch
- Data Preprocessing
- Missing Value Handling
- One-Hot Encoding
- Feature Normalization
- Gradient Descent Optimization
- Binary Cross Entropy Loss
- Classification & Regression Metrics
- Cost Function Visualization
- Interactive Loan Default Prediction

---


## 📊 Dataset

The project uses a public **Loan Default Dataset** with **148,000+ records**.

**Selected Features**
- Loan Amount
- Income
- Credit Score
- Property Value
- LTV
- Debt-to-Income Ratio
- Interest Rate
- Loan Term
- Age
- Gender
- Loan Purpose
- Occupancy Type

**Target Variable**
- **0** → Non Default
- **1** → Default
